/* 현재 폴더의 내용 리스트 조회 */
> ls

/* 현재 위치 조회 */
> pwd

/* 폴더 이동 */
> cd 이동경로               ex) cd document

/* 현재 위치 또는 파일 열기 */
> open .

/* 폴더 생성 */
> mkdir 폴더 이름           ex) mkdir tutorial => tutorial이라는 폴더 생상

/* 파일 생성 */
> touch 파일이름            ex) touch test.txt => text.txt 라는 파일 생성

/* 파일, 폴더 이동 활용하여 이름을 바꾸거나 생성도 가능 */
> mv 파일이름 이동하고싶은곳(파일 이름을 입력하면 생성또는 이름 바꾸기 가능 ) 

/* 파일 삭제 */
> rm 파일이름               ex) 폴더 삭제하고 싶다면 -r 옵션 추가 rm -r 폴더이름
